window Object:

The window object is the top-level object in the browser's JavaScript object model (DOM).
It represents the browser window or tab in which your web page is loaded.
It provides access to various properties and methods related to the browser window, such as dimensions, location, navigation, and timing.
Common properties and methods of the window object include window.location (current URL), window.alert() (displays a dialog box), window.innerWidth (viewport width), and window.setTimeout() (schedules a function to run after a specified time).

screen Object:

The screen object represents the user's screen or display device.
It provides information about the user's screen, including dimensions, color depth, and pixel density.
Common properties of the screen object include screen.width (screen width in pixels), screen.height (screen height in pixels), and screen.pixelDepth (color depth in bits per pixel).

document Object:

The document object represents the web page's HTML document that is currently loaded in the browser.
It provides access to the structure and content of the web page, allowing you to manipulate and interact with the DOM.
You can use the document object to access elements, modify their content, and respond to user events.
Common methods of the document object include document.getElementById() (retrieve an element by its ID), document.querySelector() (retrieve the first matching element), and document.createElement() (create a new HTML element).
